# Test Plan - E-commerce Application Testing

## Objective
To validate the functionality, usability, and reliability of an e-commerce platform by performing manual and API testing

## Scope
Modules to be tested:
-User Authentication (Login/Signup)
-Product Details Page
-Cart Management
-Checkout Process

## Testing Types
-Functional Testing
-Negative Testing
-UI Testing
-API Testing

## Tools Used
-Postman
-Browser (Manual Testing)
- Excel (Test Cases & Bug Reports)

## Out of Scope
- Payment gateway real transactions
